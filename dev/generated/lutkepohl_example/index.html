The following reproduces the example introduced in Section 3.2.3 of Lütkepohl (2006)

julia> using VectorAutoregressions

julia> using CSV, DataFrames, DataFramesMeta

julia> using Dates

julia> exdir = pkgdir(VectorAutoregressions) * "/examples";

julia> datadf = DataFrame!(CSV.File(exdir * "/lutkepohl_data.csv"));

julia> first(datadf, 6)
6×4 DataFrame
 Row │ date        invest  income  cons
     │ Date…       Int64   Int64   Int64
─────┼───────────────────────────────────
   1 │ 1960-03-01     180     451    415
   2 │ 1960-06-01     179     465    421
   3 │ 1960-09-01     185     485    434
   4 │ 1960-12-01     192     493    448
   5 │ 1961-03-01     211     509    459
   6 │ 1961-06-01     202     520    458

Take the subset from 1960q1 to 1978q4, and use log-difference for each series

julia> Y = @linq datadf |>
             where(:date .< Date(1979, 3, 1)) |>
             select(
                 investment = diff(log.(:invest)),
                 income = diff(log.(:income)),
                 consumption = diff(log.(:cons)),
             );
ERROR: ArgumentError: length 75 of vector returned from function #2 is different from number of rows 76 of the source data frame.

julia> first(Y, 6)
ERROR: UndefVarError: Y not defined

Calculate lag selection criteria

julia> ls = LagSelection(Y, 8)
ERROR: UndefVarError: Y not defined

Estimate the VAR

julia> v = VAR(Y, ls.selection["AIC"])
ERROR: UndefVarError: ls not defined

Check stability of the VAR

julia> sc = StabilityCheck(v)
ERROR: UndefVarError: v not defined

Check for autocorrelation in the residuals

julia> pt = PortmanteauTest(v, 12)
ERROR: UndefVarError: v not defined

julia> bg1 = LMCorrTest(v, 4)
ERROR: UndefVarError: v not defined

julia> bg2 = LMCorrTest(v, 4, smallsample = true)
ERROR: UndefVarError: v not defined
