This file reproduces the example introduced in Section 3.2.3 of Lütkepohl (2006)

julia> using VectorAutoregressions

julia> using CSV, DataFrames, DataFramesMeta

julia> using Dates

julia> exdir = pkgdir(VectorAutoregressions) * "/examples";

julia> datadf = DataFrame!(CSV.File(exdir * "/lutkepohl_data.csv"))

             select(
                 investment = diff(log.(:invest)),
                 income = diff(log.(:income)),
                 consumption = diff(log.(:cons)),
             )

Check stability of the VAR

julia> sc = checkstable(v)
VarStabilityCheck
VAR is stable
Modulus of eigenvalues (dropping repeated values):
[0.371, 0.492, 0.551, 0.57]

Check for autocorrelation in the residuals

julia> pt = portmanteau_test(v, 12)
